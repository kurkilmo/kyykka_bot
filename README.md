# kyykka_bot
Fork of [Iljinisa/kyykka_bot](https://github.com/Iljinisa/kyykka_bot/tree/main)

## Developement environment
`.env`:
```
TG_TOKEN=""
CHAT_ID=""
```

Install dependencies with
```
poetry install
```

Enter virtual environment with
```
$(poetry env activate)
```

Run application with
```
python3 kyykka_bot/main.py
```
